# SEO Expansion Plan: High-End Charlotte Neighborhoods, Lake Norman & High-Intent Keywords

## Context: Why You Rank #1 for "Best Roofing Company Near Me"

Your #1 ranking is driven by a powerful combination of signals:

1. **Brand Name Advantage**: "Best Roofing Now" literally contains "best roofing" - Google sees this as a strong brand-keyword match
2. **Dedicated Keyword Page**: `/best-roofing-company-charlotte-nc` with exact-match title, 50+ keyword variations
3. **Massive Topical Authority**: ~1,320+ pages covering every roofing topic imaginable creates domain-wide authority
4. **Rich Schema Markup**: LocalBusinessSchema, AISearchOptimizationBundle, VoiceSearchActionSchema, ReviewsSchema, FAQSchema - you're speaking Google's language
5. **E-E-A-T Signals**: 5.0 Google rating (62 reviews), BBB A+, veteran-owned, 500+ roofs, CertainTeed/GAF certified
6. **Geo Signals**: CityGeoSection on 197 pages with Maps embeds + 95+ directory citations
7. **Homepage Authority**: Strong brand keywords + trust signals in title/meta

### The Replicable Pattern
Every page that ranks well follows this formula:
- Exact-match keyword in title (under 60 chars)
- Keyword-rich meta description with trust signals (rating, review count, BBB)
- Schema markup (LocalBusiness + FAQ + AI/Voice bundles)

## Implementation Details Per Page

### Template Pattern (Proven Formula)
Each new page should follow the existing pattern from successful pages:

```tsx
// 1. Metadata with exact-match title (<60 chars) + keyword-rich description
export const metadata: Metadata = {
  title: 'Target Keyword | Trust Signal',
  description: 'Keyword + trust signals (rating, reviews, BBB, veteran-owned) + CTA',
  keywords: [/* 10-15 keyword variations */],
  alternates: { canonical: `${SITE_CONFIG.url}/slug` },
  openGraph: { /* ... */ },
};

// 2. Schema markup
<LocalBusinessSchema />
<BreadcrumbSchema />
<FAQSchema />
<AISearchOptimizationBundle />
<VoiceSearchActionSchema />

// 3. Page content
<Breadcrumbs />
{/* Hero with H1 containing primary keyword */}
{/* Service-specific content sections */}
{/* FAQ section (5-8 questions targeting long-tail queries) */}
<GeoProjectGalleryStrip />  {/* Visual proof */}
<CityGeoSection />          {/* Geographic authority */}
<CTASection />               {/* Conversion */}
```

### Neighborhood Page Content Requirements
For each high-end neighborhood page:
1. **Neighborhood-specific H1**: "Roofing in [Neighborhood] Charlotte NC"
2. **Local knowledge section**: Home ages, typical roof types, HOA requirements
3. **Premium materials section**: What materials suit the neighborhood's architectural style
4. **Nearby landmarks**: Schools, country clubs, parks for geo-relevance
5. **Project gallery**: Filtered to that area if available
6. **FAQ**: "How much does a roof cost in [Neighborhood]?", "Best roofing material for [Neighborhood] homes?"
7. **Internal links**: To luxury-roofing page, material pages, nearby neighborhood pages

## Internal Linking Strategy

### Hub-and-Spoke Model
- **Charlotte luxury hub**: `/luxury-roofing-charlotte-nc` links to all high-end neighborhood pages

- **Material hubs**: `/copper-roofing-charlotte-nc`, `/slate-roofing-charlotte-nc` link to luxury + neighborhood pages where those materials are common

### Cross-Link New Pages
Each new neighborhood page should link to:
1. The city location page (e.g., `/roofing-charlotte-nc`)
2. The luxury roofing hub page
3. 2-3 relevant service pages (roof replacement, repair, inspection)
4. 2-3 nearby neighborhood pages
5. Material pages relevant to that neighborhood's architecture
